# Free Art APIs

| Source | Best for | Access |
|---|---|---|
| The Metropolitan Museum of Art | Very broad: ancient art, European painting, Asian, African, Islamic, decorative arts, photography | Public REST API; no key required |
| Rijksmuseum | Dutch masters, prints, drawings, photography, decorative arts, historical objects | Search, Linked Data, IIIF and harvesting APIs; current search API needs no key |
| Smithsonian Open Access | Art, design, photography, crafts, posters, sculpture, cultural objects and some 3D material | Public API with free API-key registration; downloadable JSON datasets |
| Victoria and Albert Museum | Design, fashion, textiles, jewellery, ceramics, posters, illustration, photography and decorative art | Public REST API plus IIIF image service; apparently no key for normal use |
| Art Institute of Chicago | Classical through modern art, prints, photography, design, architecture and contemporary work | Public REST API; no key generally required |
| Cleveland Museum of Art | Global art from antiquity to modernism, including Asian, African, Islamic and decorative arts | Public API and bulk datasets; no key |
| Harvard Art Museums | Very broad academic collection: ancient, European, Asian, Islamic, prints, drawings, photography, modern and contemporary | REST API; free API key required |
| Europeana | Cross-institutional European art, photography, design, manuscripts, posters, sound, film and cultural heritage | Search, Record and IIIF APIs; free key required |
| Wikimedia Commons | Aggregated public-domain and Creative Commons art, photography, diagrams, museum scans and digital media | MediaWiki API, Wikidata SPARQL and Commons category APIs; no key |
| Flickr Commons | 100+ institutions (British Library, Wellcome Collection, NYPL, national archives, etc.) donating public-domain collections — different institutions than our other sources | Free API key at flickr.com/services/api/ |
| Unsplash | Contemporary creative photography and illustration rather than museum art | Free developer registration; up to 1,000 requests per hour on the published free tier |
